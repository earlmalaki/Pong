# Pong

A basic PingPong game made using Java.

Left racquet represents Player 1.
Right racquet represents Player 2.

A machine problem (MP #4) submission in CMSC22 - Object Oriented Programming

### Controls

Player 1:

  W - Move Up
  
  S - Move Down
  
Player 2:

  Arrow UP - Move Up
  
  Arrow Down - Move Down
